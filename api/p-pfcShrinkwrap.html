Package com.ptc.pfc.pfcShrinkwrap

Description

Contains interfaces for shrinkwrap export.

Interfaces
ShrinkwrapFacetedFormatInstructionsThis interface contains instructions for exporting a solid model to a faceted shrinkwrap format.
ShrinkwrapFacetedPartInstructionsThis interface contains instructions to export a solid model to a shrinkwrap faceted part.
ShrinkwrapMergedSolidInstructionsThis interface contains instructions to output a solid model to a shrinkwrap model while merging components.
ShrinkwrapModelExportInstructionsThis interface contains the instructions used for export of a solid model to a shrinkwrap format.
ShrinkwrapSTLInstructionsThis interface contains instructions to export a shrinkwrap STL file.
ShrinkwrapSurfaceSubsetInstructionsThis interface contains instructions to export a shrinkwrap model using a subset of surfaces.
ShrinkwrapVRMLInstructionsThis interface contains instructions to export a model to a shrinkwrap VRML file.
