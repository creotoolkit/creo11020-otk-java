Class ComponentConstraints

package com.ptc.pfc.pfcComponentFeat;

public class ComponentConstraints
  implements jxobject


Description

This interface contains a sequence of assembly component constraints.

The user can access elements by their integer index (position) in the list. Sequence provides methods for positional (indexed) access to its elements. Sequences are zero based.
