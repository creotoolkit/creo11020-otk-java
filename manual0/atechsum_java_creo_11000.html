Technical Summary of Changes for Creo 11.0.0.0
This section describes the critical and miscellaneous technical changes in Creo 11.0.0.0 and Creo Object TOOLKIT Java. It also lists the new and superseded functions for this release.
New Functions
This section describes new functions for Creo Object TOOLKIT Java for Creo 11.0.0.0.
Kinematic Dragging
New Function
Description
wfcSession.WSession.StartKinDrag
Starts a drag session.
wfcSession.WSession.StopKinDrag
Stops the drag session.
wfcSession.WSession.SetKinDragEnvironment
Sets the environment for dragging and reconnect operations.
wfcSession.WSession.MoveKinDrag
Drags the selected geometric object to the specified X and Y screen coordinates.
wfcSession.WSession.MoveKinDragForModel
Brings to drag the selected object.
wfcSession.WSession.CreateSnapshot
Creates a new snapshot for the top-level assembly as per its current position in the Creo Parametric window
wfcSession.WSession.UpdateSnapshot
Updates the snapshot for the top-level assembly as per its current position in the Creo Parametric window.
wfcSession.WSession.DeleteSnapshot
Deletes a specified snapshot.
wfcSession.WSession.GetKinSnapshotTransformations
Retrieves the transformation paths and transformation matrices saved in the snapshots for the subassemblies and their components with respect to the top-level assembly snapshot.
